# ZetaAI
ZetaAI is an AI model trained by NeuralNexusLab.


**Calling API:**

1. JavaScript (Browser & Node.js)
```javascript
fetch("https://zeta.nethacker.cloud/api?prompt=hello")
  .then(res => res.json())
  .then((data) => {
    console.log(data.return)
  });
```

2. CLI
```bash
curl "https://zeta.nethacker.cloud/api?prompt=hello"
```
